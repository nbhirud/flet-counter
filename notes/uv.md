
# uv

## Install uv 

### standalone installers

https://github.com/astral-sh/uv?tab=readme-ov-file#installation


#### On macOS and Linux.
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

#### Update standalone installation
```
uv self update
```

### PyPI

#### With pip
```
pip install uv
```
#### Or pipx
```
pipx install uv
```
